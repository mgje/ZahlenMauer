Zahlenmauer

Mit Hilfe dieser Webanwendung lassen sich beliebige Zahlenmauern innerhalb eines Webbrowser darstellen.

Um erste Erfahrungen zu CoffeeScript (sehr effiziente Sprache für JavaScript Programmierung) zu sammeln, erstellte ich im Rahmen eines 90 Minuten Projektes diese einfache Webanwendung. 

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

Installation

  • Archiv herunterladen (zip)
  • Zip-Datei auspacken
  • die Datei zahlenmauer.html im Verzeichnis public in einem Browser öffnen
